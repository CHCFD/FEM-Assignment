Two files are required to run the code and they are
    - main.py
    - class_helmholtz.py

The code has been tested on Python 3.7.4

main.py
- main script that runs the solution using class helmholtz

class_helmholtz.py
- Contains member function to solve the helmholtz problem
- Functions that construct/solve global/elemental matrices

How to run the code?
- On the terminal (linux) $ python main.py
- Alternative, you can run on Spyder IDE (tested)
- The code allow the user to run specified element types or no. of elements
    which can be changed in main.py
